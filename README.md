# Smart-Parking-System
Tugas besar sistem tertanam 2020-2021 ITERA
Kontrol dan monitoring parkiran berbasis web server Adafruit IO
Komponen yang digunakan:
- arduino UNO
- nodemcu esp8266
- IR obstacle sensor
- Ultrasonic sensor
- LCD 16x2
- LDR
- Servo
- LED dan Buzzer
